## Overview

This document analyzes the Austrian elementary school mathematics curriculum (Volksschule Mathematik) to guide the development of the "Mathematik" app within the mini-trainer-engine architecture.

### Educational Objectives (Bildungs- und Lehraufgabe)

The mathematics curriculum aims to enable students to:

1. **Creative Activity** - Engage in playful, research-discovery, and constructive activities
2. **Rational Thinking** - Develop logical thinking and problem-solving behavior
3. **Practical Application** - Experience the practical usefulness of mathematics
4. **Technical Skills** - Acquire fundamental mathematical techniques

### Core Subject Areas

The curriculum is divided into four main areas (plus one additional for Grundstufe II):

| Area | German Term | Description |
|------|-------------|-------------|
| Natural Numbers | Aufbau der natürlichen Zahlen | Number concepts, place value, number sense |
| Operations | Rechenoperationen | Addition, subtraction, multiplication, division |
| Quantities/Measurements | Größen | Length, weight, time, money, area |
| Geometry | Geometrie | Spatial reasoning, shapes, measurements |
| Fractions | Bruchzahlen | Only in Grundstufe II (4th grade) |

## Level Structure

### Grundstufe I (1st & 2nd Grade)

#### 1. Schulstufe (1st Grade)

**Aufbau der natürlichen Zahlen (Numbers)**

| Topic | Description |
|-------|-------------|
| Basic Mathematical Abilities | Properties, distinguishing, comparing, assigning, ordering, classifying |
| Number Concept Development | Gaining number concepts including zero (counting, simultaneous recognition) |
| Number Space to 20 | Action-oriented representation and structuring |
| Number Representations | Number images, quantity representations, number line, symbolic representation |
| Orientation | Building number sequences, relations using symbols (=, ≠, <, >) |
| Decimal Structure | Understanding place value, digits vs. place value |
| Operative Exploration | Finding neighboring numbers, additive/multiplicative decomposition |

**Rechenoperationen (Operations)**

| Topic | Description |
|-------|-------------|
| Understanding Operations | Actions: adding, removing, completing, multiplying, dividing, measuring |
| Operation Structures | Using terms: plus, minus, equals, times, divided by, contained in |
| Symbols | +, -, =, ·, : |
| Additive Operations | Number space to 20, addition/subtraction tables |
| Multiplicative Operations | Number space to 20, multiplication tables (1x1) |
| Application | Word problems, game situations |

**Größen (Measurements)**

| Topic | Description |
|-------|-------------|
| Length | Comparing lengths, measuring with arbitrary units |
| Weight | Comparing weights, using balance scales |
| Time | Day/week structure, clock reading (full hours, half hours) |
| Money | Euro coins and bills, simple purchases |

**Geometrie (Geometry)**

| Topic | Description |
|-------|-------------|
| Spatial Orientation | Position terms (above, below, beside, etc.), left/right |
| Directions | Describing paths, directional changes |
| Basic Shapes | Cubes, cuboids, circles, squares, triangles |
| Creative Design | Building, laying, folding, cutting |

## Didactic Principles

The curriculum emphasizes these teaching principles that should inform exercise design:

### 1. Operative Building and Working Through

- Learning through actions with varied materials
- Emphasizing problem representations
- Research-based, experimental approach
- Uncovering different solution paths
- Highlighting connections between operations

**Exercise Implications:**

- Provide multiple solution methods
- Show connections between related problems (e.g., 3+4=7 and 7-3=4)
- Include "reverse" problems (Tausch-, Nachbar-, Umkehraufgaben)

### 2. Functions of Practice

- **Operative Practice**: Building calculation abilities through understanding
- **Skill Practice**: Automating basic facts and techniques

**Exercise Implications:**

- Include both conceptual and drill exercises
- Use manageable number ranges
- Avoid purely mechanical repetition

### 3. Variation of Visualization

- Avoid binding concepts to single materials
- Show fractions as parts of objects, areas, quantities, sets

**Exercise Implications:**

- Use diverse visual representations
- Vary contexts for same concepts
- Include multiple image types

### 4. Variation of Representation Levels

- Switch between: enactive (action), iconic (image), symbolic

**Exercise Implications:**

- Include concrete object images
- Use symbolic representations
- Connect visual to abstract

### 5. Mathematical Variation

- Vary non-essential attributes to highlight essential ones
- Change position, color, size while keeping key properties

**Exercise Implications:**

- Show shapes in different orientations
- Use varied colors and sizes
- Maintain focus on mathematical properties

## Conclusion

The Austrian elementary mathematics curriculum provides a comprehensive framework for developing mathematical understanding from basic number concepts to complex problem-solving. The mini-trainer-engine's existing exercise types can effectively support most curriculum requirements, with potential extensions for geometry and measurement topics.

Key success factors for the Mathematik app:

1. **Progressive Difficulty** - Align with curriculum's number space progression
2. **Multiple Representations** - Use varied visual and symbolic forms
3. **Connected Operations** - Show relationships between operations
4. **Practical Context** - Include real-world word problems
5. **Differentiated Practice** - Balance conceptual understanding with skill fluency
